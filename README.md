# oilmonitoring
Fuel Station Monitoring, Reporting, and Dashboard System for Ministry Reporting  ระบบเว็บสำหรับบันทึก จัดเก็บ ตรวจสอบ และแสดงผลข้อมูลที่เกี่ยวข้องกับสถานีบริการน้ำมัน เช่น พิกัดปั๊มน้ำมัน, ยี่ห้อปั๊มน้ำมัน, ชนิดน้ำมัน, ปริมาณการเติมสูงสุด, ปริมาณน้ำมันสะสม, ดยมี Dashboard และรายงานสรุปเพื่อสนับสนุนการบริหารจัดการและการรายงาน
